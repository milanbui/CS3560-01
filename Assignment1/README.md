# Assignment 1 | iVote Simulator | Due 29 Sept 2020
Description:     
Mimics a voting service like kahoot. In the driver, a question and candidate answers are created. The question can be configured to be either multiple choice or single choice. The candidate answers can be marked as a correct answer. The voting service must be created using the information mentioned before and will accept answer submission afterwards. It can also display the results of the votes.
